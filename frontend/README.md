# WildCard Loss Analyzer

투자 손실을 분석하고 맞춤형 학습 경로를 제공하는 AI 기반 투자 분석 도구입니다.

Upstage Solar Pro 2를 활용하여 투자 패턴을 분석하고 개선 방안을 제시합니다.

## 기술 스택

- **Frontend**: React + TypeScript + Vite
- **UI**: Tailwind CSS
- **AI Model**: Upstage Solar Pro 2

## 로컬 실행 방법

**사전 요구사항:** Node.js

1. 의존성 설치:
   ```bash
   npm install
   ```

2. 환경 변수 설정:
   프로젝트 루트에 `.env.local` 파일을 생성하고 Upstage API 키를 설정합니다:
   ```
   UPSTAGE_API_KEY=your_api_key_here
   ```

3. 앱 실행:
   ```bash
   npm run dev
   ```

4. 브라우저에서 `http://localhost:5173` 접속

## 주요 기능

- 📊 투자 손실 원인 분석
- 🔍 기술적 시장 상황 분석
- 📚 맞춤형 학습 경로 제안
- 💡 행동 투자 가이드 제공
- 💬 AI 투자 어드바이저와 실시간 대화
